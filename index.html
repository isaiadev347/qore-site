What is QORE?

QORE is a control plane for AI runtime pricing, diagnostics, route integrity, and NTU-based economics measurement. It sits above fragmented AI provider pricing and model ecosystems—normalizing route logic, cost representation, and usage accounting.

One API key across all supported providers. One wallet. Full visibility into requested-vs-actual provider and model, with drift detection and confidence scoring.

Compute Unit

What is NTU?

NTU (Normalized Token Unit) is QORE's internal unit for expressing AI compute economics across incompatible provider pricing schemes—tokens, images, seconds, runs.

Providers all meter differently. NTU acts as a vendor-neutral compute-value metric that normalizes cost and work across AI models and modalities—LLM calls, image generation, audio, embeddings, and more.

NTU is part of the broader QORE project logic, not just a cosmetic label. It enables a single budget and routing policy across your entire AI stack.

NTU is not a token, currency, cryptocurrency, or financial asset of any kind.

The Problem

Why QORE?

AI infrastructure today is fragmented. QORE consolidates the complexity into a single control surface.

API

Universal API Access

Managing separate API keys, separate billing dashboards, and separate rate limits for every AI provider is operational overhead. QORE gives you one key across providers—one place to fund, monitor, and control your AI spend.

INT

Route Integrity

AI providers can silently substitute a cheaper or different model than the one requested. QORE surfaces these substitutions: it compares requested-vs-actual provider and model, detects cost and latency anomalies, and classifies findings by severity—from no_issue to premium_integrity_violation.

Clarity

What QORE is not

Not a Model

QORE is not an AI model

QORE is a routing, pricing, and accounting layer for AI infrastructure. It does not generate AI responses—it routes requests to external providers.

No Training

No customer data training

QORE does not use customer data to train any model because QORE contains no model. Routing decisions are based on system logic, not machine learning on user content.

Not Crypto

NTU is not a cryptocurrency

NTU is purely an internal measurement unit used inside QORE software. It is not a token, currency, cryptocurrency, or financial asset of any kind.
